Aplicación de registro y venta de chocolates
Permite almacenar en una base de datos* los distintos tipos de chocolates que existen para vender
Tiene las funciones de agregar, eliminar y modificar registros, así como mostrarlos y buscar por código

*En este caso es una base de datos local MySQL. Para poder conectarse es necesario crear una base de datos en MySQL workbench,y ese nombre
agregarlo en la clase "Conexión" junto con la password de MySQL del usuario.
Se puede habilitar para ser usada en oracle o remoto.
